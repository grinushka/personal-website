# Personal Website

This is my personal corner of the web, crafted with React and TypeScript. It's got a neat dark/light mode switch, a pretty aurora animation in the background (yeah, like the real thing!), and a cool typewriter effect that tells you a bit about me. Simple, clean, and hopefully fun to look at!


## Tech Stack
- React + TypeScript
- TailwindCSS + Shadcn UI
- TanStack Router
- Biome for formatting and linting

## Getting Started

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

## Building For Production

Build the project:
```bash
npm run build
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run format` - Format code with Biome
- `npm run lint` - Check and fix linting issues
